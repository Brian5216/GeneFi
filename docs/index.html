Core Features

🏭

自适应进化 Adaptive Evolution

8种市场体制自动检测，策略参数通过遗传算法实时进化，持续适应市场变化。

🤖

多智能体 A2A 协议

3个AI Agent自治协作：Predictor预测市场→Executor执行策略→Judge裁判适应度。

🔗

OKX Agent Trade Kit (MCP)

通过MCP协议接入119个OKX交易工具，支持合约/现货/期权/网格/算法单。

📊

蒙特卡洛统计验证

多体制回测+t检验，科学证明进化策略产生超额收益(Alpha)。

🛡

安全模式 Safe Mode

种群适应度连续下降自动切换OKX Earn稳健理财，仿生"休眠"避险。

🚀

一键部署 Deploy

进化最优策略一键通过MCP部署到OKX Demo Trading，获得真实orderId。
